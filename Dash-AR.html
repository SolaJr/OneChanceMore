Dash AR

Welcome to a virtual community connected by nothing except for Augmented Reality and a desire for living healthy.

 

Getting people to try fitness app for the first time is a fairly simple task. Getting them to try it a secound or third times is where the true problem lies! This mobile AR App creates a virtual world where users can race against their friends and themselves an compete for prizes such as a month of free Netflix.

Dash - Ar

November 2017 to February 2018

Position : Designer, Co Producer, Pitcher, and Researcher

Sponsor : Startup Aggieland

Industries : Product Design, Mechanical Engineering, User Experience

Need Statement : By using Virtual Reality or Augmented Reality, create a product that can encourage continued healthy living. The end result was an Augmented Reality program that would allow a user to make the practice of running or walking more enjoyable by transforming it into a game with features allowing you to race your friends, run from zombies, or simply race past versions of yourself.

Contributions  

Designed the program in code in C# and Unity with a Meta Vision Headset

Planned and organized group meetings
